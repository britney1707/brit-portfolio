Projet menu deroulant

1. Utilse une balise nav pour contenir le menu. 2.Cree une liste ul avec des elements li pour chaque lien. 3.Pour le menu deroulant, imbrique une autre liste ul dans un li.4.En CSS cache la sous liste par defaut avec display none. 5.Affiche-la au survol avec li:hover ul{ display:block;}

projet texte sur une image

1.Place l'image et le texte dans une meme div avec position: relative;. 2.Applique position:absolute; au texte pour le positionner par-dessus l'image. 3.Utilise top,left,right ou bottom pou placer precisement le texte. 4.Utilise color,font-size, background,etc pour styliser le texte. 5.Assure toi que l'image est bien visible avec width: 100%;.

projet photo grid

1.Place toutes les images dans un div conteneur 2.Applique display grid au conteneur pour activer la grille 3.Utilise grid-template-columns: repeat(auto-fit,minmax (200px,1fr)); pour creer une grille responsive. 4.Ajoute gap 10px pour espacer les images 5.Utilise width 100% sur les images pour quelles s'adaptent
